# table percentage

*Diamond anatomy*

**Table percentage is the width of a diamond's table facet divided by its average girdle diameter, expressed as a percentage. It describes how large the top window is relative to the whole stone.**

Table percentage describes how big a diamond's top window is relative to the whole stone. The formula is the width of the table facet divided by the average girdle diameter, multiplied by one hundred. On a round brilliant the table width is measured straight across the eight-sided table.

The number captures a balance. The flat table returns bright light straight up through the center of the stone, while the smaller angled crown facets around it split light into color. A larger table opens up that bright central return but crowds out the crown facets, so a very high table percentage can read as bright but a little flat and short on fire. A smaller table leaves more room for fire from the crown but shrinks the central window. Because brightness and fire pull against each other, a grading lab weighs table percentage as one factor in the stone's overall proportions rather than judging it on its own, and the flattering range shifts from one shape to another.

In practice, table percentage is always read together with depth percentage, crown angle, and pavilion angle, since those proportions work as a set. Marcel Tolkowsky's 1919 proportions for the round brilliant included a specific table size, one of the early efforts to describe the balance that makes the cut look lively.
